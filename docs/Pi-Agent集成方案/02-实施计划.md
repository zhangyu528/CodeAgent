# 实施计划：Pi Agent Core 直接集成

本项目将直接使用 `@mariozechner/pi-agent-core` 的原生 SDK 重新实现 CLI 逻辑，跳过对旧代码的适配。

## 实施阶段

### 第一阶段：基础设施 (Infrastructure)
1.  安装依赖：`@mariozechner/pi-agent-core` 和 `@mariozechner/pi-ai`。
2.  初始化 `ModelRegistry` 并集成现有环境变量。

### 第二阶段：工具重构 (Tools)
1.  按照 Pi 标准重新定义工具（基于 Zod 校验）。
2.  目前已实现：`read_file`, `write_file`, `run_command`, `list_directory`。

### 第三阶段：UI 与核心集成 (Integration)
1.  重构 `src/apps/cli/ink/app.tsx` 以监听 Session 事件。
2.  实现 Token 流式显示和工具生命周期管理。

### 第四阶段：独立入口 (Entry Point)
1.  创建 `pi_index.ts` 作为新核心的独立启动文件，验证 MVP 功能。
