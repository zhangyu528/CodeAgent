# 架构分析：引入 Pi Agent Core

将当前的 `AgentController` 和 `LLMEngine` 替换为 `@mariozechner/pi-agent-core` 这是一个重大的架构决策。以下是对该方案的优缺点分析及建议。

## 执行摘要
Pi Agent Core 是一个专门为构建响应式 AI 代理而设计的、高度专业化的事件驱动框架。它提供了先进的交互模式（如转向指令 Steering、后续指令 Follow-up）和清晰的关注点分离。

---

## 核心对比表

| 特性 | 当前 CodeAgent 核心 | Pi Agent Core |
| :--- | :--- | :--- |
| **LLM 支持** | 自定义 `LLMEngine` | `@mariozechner/pi-ai` (支持广泛) |
| **代理循环** | 顺序执行循环 | 事件驱动，可配置 |
| **UI 集成** | `IUIAdapter` 钩子 | 原生事件流支持 |
| **交互能力** | 基础中断处理 | 支持 Steering 和 Follow-up |
| **状态/会话** | 线性会话 | 树状会话 (Branching) |

---

## 核心优势
1.  **响应式架构**：完美的事件驱动模式，极大简化 UI 与核心的交互。
2.  **会话树状管理**：原生支持分支和回溯，容错率更高。
3.  **标准化扩展**：通过 Extension 机制实现解耦，符合开闭原则。
