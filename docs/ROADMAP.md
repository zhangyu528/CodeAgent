# CodeAgent Feature Roadmap & 看板

本文档记录了 CodeAgent 从“引擎研发”转向“功能驱动 (Feature-driven)”后的路线图。每个 Feature 都会经历从“规划”到“里程碑达成”的全生命周期。

## 核心演进路线

| 优先级 | 功能 ID | 功能名称 | 描述 | 状态 | 相关文档 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **High** | F1 | **源码全局感知** | 支持 codebase 级别的语义搜索与符号理解。 | ✅ 已完成 | [详情](功能需求/F1_源码全局感知.md) |
| **High** | F2 | **Git 全自动化** | 自动 commit, 自动分支管理, 生成高质量 PR 说明。 | 📅 待启动 | [详情](功能需求/F2_Git全自动化.md) |
| **Med** | F3 | **环境自动上下文** | 启动时自动读取 README/JSON/Tree 注入上帝视角。 | ✅ 已完成 | [详情](功能需求/F3_环境自动上下文.md) |
| **Med** | F4 | **多模型多 Provider** | 支持集成 DeepSeek, OpenAI 以及动态模型切换。 | ✅ 已完成 | [详情](功能需求/F4_多模型多Provider.md) |
| **Low** | F5 | **浏览器增强** | 具备 Web Search 能力，查询最新文档与错误。 | ✅ 已完成 | [详情](功能需求/F5_浏览器增强.md) |
| **High** | F6 | **工作区授权** | 启动时提示信任当前目录，防误触。 | ✅ 已完成 | [详情](功能需求/F6_工作区授权.md) |
| **High** | F7 | **CLI 交互式会话** | 支持流式输出、Token滑窗记忆与多行输入的高效终端对话模式。 | ✅ 已完成 | [详情](功能需求/F7_CLI交互式会话.md) |
