# Feature 需求：F8 - CLI 高级交互增强 (Advanced CLI Interaction)

## 1. 背景与目标
在完成 F7 基础流式交互后，为了进一步提升开发者在终端的操作效率与直观感受，需要引入更高级的 UI 组件与交互逻辑。目标是让 CodeAgent 的终端界面从简单的“打字机”进化为具备“可视化反馈”与“专业开发工具感”的高级界面。

## 2. 核心功能点

### 2.1 代码差分可视化 (Rich Diff Rendering)
- **功能**：在提议修改文件前，以彩色的 Git-style Diff 格式展示变更逻辑。
- **效果**：使用红色 `-` 标示删除，绿色 `+` 标示新增，并保留上下文行。

### 2.2 交互式选择菜单 (Interactive Selection)
- **功能**：在遇到多选场景（如选择文件、确认分支、选择修复策略）时，展示可选列表。
- **效果**：支持上下方向键选择、空格勾选、或输入关键词自动过滤。

### 2.3 终端编辑器模式 (Embedded Editor Mode)
- **功能**：处理长文本输入（如复杂的任务描述或长段代码粘贴）。
- **效果**：支持调起系统默认编辑器（Vim/Nano/VSCode）进行编辑，保存后内容自动返回 Agent。或实现简易的块输入模式（`<< EOF`）。

### 2.4 工具执行状态折叠 (Tool Execution Bubbles/Progress)
- **功能**：解决 Agent 连续调用工具时的内容刷屏问题。
- **效果**：将工具执行过程展示为一行带 Spinner 的进度条，执行完后变为绿色的 ✅，并支持点击或命令展开查看详情。

### 2.5 高级安全确认面板 (Enhanced Security Prompt)
- **功能**：强化高危操作（如 `rm -rf`）的警示感。
- **效果**：在确认框中突出显示待执行命令，并增加风险等级颜色标识（如红色加粗）。

### 2.6 终端快捷键支持 (Global Keybindings)
- **功能**：无需输入命令即可触发常用操作。
- **效果**：
  - `Ctrl + L`：清屏且重置 Session。
  - `Ctrl + C`：中断当前生成。
  - `Tab`：文件名或命令补全。

## 3. 验收标准
- [ ] 修改代码时，用户能直接在 CLI 看到直观的彩色 Diff 块。
- [ ] 出现多文件候选项时，用户可以使用方向键进行交互选择。
- [ ] Agent 执行一系列搜索与文件操作时，界面不再大面积翻滚，而是展示紧凑的进度列表。
- [ ] 支持通过快捷键快速重置会话。

## 4. 技术方案
- **Diff 渲染**：引入 `diff` 库结合 `chalk` 进行着色。
- **交互菜单**：深化使用 `@inquirer/prompts` 的 `select` 和 `checkbox`。
- **状态展示**：引入 `log-update` 或类似库实现正在执行行的原地更新和折叠逻辑。
- **快捷键**：监听 `process.stdin` 的 `keypress` 事件。
