# CLI REPL 规范（持久记忆 + 流式响应 + 基本指令）

本规范用于定义终端 REPL 对话界面的行为与用户体验。

## 目标

- 持久记忆：每轮对话写入 VectorStore
- 流式响应：逐 token 打印
- 控制指令：`/help`、`/exit`、`/reset`

## 指令定义

### `/help`
显示可用指令与用法示例。

### `/exit`
退出 REPL。

### `/reset`
重置当前会话上下文（清空短期记忆），保留长期记忆（VectorStore）。

## 交互行为

1. 用户输入一行
2. 若匹配指令，执行对应行为
3. 否则调用 Agent 执行并流式输出结果
4. 每轮对话结束，将 user/assistant 内容写入 VectorStore

## 输出规范

- 流式输出逐 token 显示
- 每轮完成后换行
- 发生错误时输出可读错误信息
