# 技术预览：新架构代码片段

### 1. 核心工厂 (factory.ts)
```typescript
import { ModelRegistry, AuthStorage } from '@mariozechner/pi-ai';
import { Agent } from '@mariozechner/pi-agent-core';

export class PiAgentManager {
  constructor() {
    const auth = AuthStorage.create();
    this.registry = new ModelRegistry(auth);
    // 自动注册模型...
    this.agent = new Agent({
      modelRegistry: this.registry,
      tools: allTools,
    });
  }
}
```
