# SwiftUI Client (Swift Package)

This folder is a Swift Package executable that launches a SwiftUI app.

## Open
- Open the folder in Xcode: `swiftui-client/`
- Xcode will treat `Package.swift` as the project.

## Run
- Product: `CodexJSONLClient`
- Click Run in Xcode or run via terminal:
  - `swift run CodexJSONLClient`

## Files
- `Sources/CodexJSONLClient/App.swift`
- `Sources/CodexJSONLClient/AgentClient.swift`
- `Sources/CodexJSONLClient/CodeAgentViewModel.swift`
- `Sources/CodexJSONLClient/ContentView.swift`
